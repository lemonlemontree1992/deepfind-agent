# 复制功能使用指南

## 📋 多种复制方式

### 方式 1：一键复制按钮（推荐）⭐
**最快捷的方式**

```
┌────────────────────────────────┐
│ 纽约4天3晚旅行攻略              │
│                                │  ← 消息内容
└────────────────────────────────┘
         [📋 复制] 你            ← 点击按钮一键复制
```

**特点**：
- ✅ 点击即可复制全部内容
- ✅ 按钮醒目，易于发现
- ✅ 视觉反馈明显（变绿色）
- ✅ 支持用户输入和 AI 回复

---

### 方式 2：鼠标选中文本复制

**适合复制部分内容**

```
操作步骤：
1. 用鼠标拖选想要复制的文本
2. 右键 → 复制
   或 按 Ctrl+C (Windows) / Cmd+C (Mac)
```

**特点**：
- ✅ 可以只复制需要的部分
- ✅ 浏览器原生支持，无需额外功能

---

## 🎨 交互友好性设计

### 视觉设计原则

1. **明显性**
   - 按钮背景色明显，易于发现
   - 图标深色，清晰可见
   - 添加边框和阴影增强识别度

2. **反馈性**
   - 点击后立即变绿色
   - 显示"已复制"文字
   - 顶部弹出成功提示

3. **易用性**
   - 按钮尺寸适中（32px 高度）
   - 点击区域充足，易于点击
   - 添加文字说明"复制"，更直观

4. **一致性**
   - 用户消息和 AI 消息都使用相同样式
   - 复制成功后的绿色主题统一
   - 提示信息位置固定（顶部居中）
