# 复制功能使用说明

## ✅ 已完成的修改

### 新增功能

1. **复制按钮组件** (`frontend/src/components/CopyButton.tsx`)
   - 一键复制消息内容到剪贴板
   - 复制成功后显示绿色对勾图标（2秒后恢复）
   - 使用 toast 提示复制状态

2. **消息气泡复制功能**
   - ✅ 用户消息：可复制自己的问题
   - ✅ Agent 消息：可复制回复内容
   - ✅ 流式输出：实时生成时可复制当前内容
   - ✅ 欢迎消息：可复制欢迎语

### 修改的文件

1. **`frontend/src/pages/Index.tsx`**
   - 导入 `Copy`, `Check` 图标
   - 导入 `CopyButton` 组件
   - 在所有消息气泡下方添加复制按钮

2. **`frontend/src/components/CopyButton.tsx`** (新建)
   - 创建可复用的复制按钮组件
   - 支持自定义样式
   - 使用 Clipboard API 复制文本

## 🎨 设计特点

- **非侵入式**：复制按钮小巧，不影响阅读体验
- **即时反馈**：复制成功后图标变为绿色对勾 ✓
- **通用性**：适用于所有消息类型（用户/Agent/流式）
- **响应式**：支持桌面和移动端

## 🚀 使用方式

1. **复制用户问题**：点击自己发送的问题下方的复制按钮
2. **复制 Agent 回复**：点击报告内容下方的复制按钮
3. **复制流式内容**：生成过程中可随时复制当前已生成的内容

## 🔧 技术实现

```typescript
// CopyButton 组件核心代码
const handleCopy = async () => {
  try {
    await navigator.clipboard.writeText(text);
    setCopied(true);
    toast.success('已复制到剪贴板');
    setTimeout(() => setCopied(false), 2000);
  } catch (err) {
    toast.error('复制失败，请重试');
  }
};
```
