## API Key申请流程

### Reddit API（免费）
1. 访问 https://www.reddit.com/prefs/apps
2. 点击"create another app..."
3. 选择"script"
4. 填写名称: DeepFindAgent
5. 填写描述: Deep research agent
6. 获取client_id和client_secret
7. 状态: ⏳ 待申请

### OpenWeatherMap API（免费）
1. 访问 https://openweathermap.org/api
2. 注册账号
3. 创建API Key
4. 免费额度: 1000次/天
5. 状态: ⏳ 待申请

### ExchangeRate API（免费）
1. 访问 https://www.exchangerate-api.com/
2. 注册账号
3. 创建API Key
4. 免费额度: 1500次/月
5. 状态: ⏳ 待申请
